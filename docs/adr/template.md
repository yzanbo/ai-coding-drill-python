## Decision（決定内容）

- **戦略判断型**（一度決めたら基本動かない。例：AWS 単独 / Drizzle 採用 / Postgres ジョブキュー）：判断本体を ADR 本文に書く。詳細表・規約・運用ルールも本セクションで完結させる
